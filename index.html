SteamWreck

2020

Back End Developer

SteamWreck is an application that uses the public profile data of a Steam user and responds with game recommendations from their unplayed games that is decided by an algorithm I developed.

The entire back end of the project was designed and implemented by myself, from the Express server, to the MongoDB Cloud database. I juggled two different API's to gather data, and fed that data into an algorithm to recommend Steam games. In order to speed up loading times, I scheduled a background worker to gather more data during off hours and made unit tests to cover all the important parts of the algorithm. Last but not least, authentication through the OpenID service Steam provides was used as a way to verify the user and access his personal recommendations.
